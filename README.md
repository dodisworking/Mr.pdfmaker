## How to Use

1. Open `index.html` in your web browser
2. Click on Mr PDF Maker to upload your file
3. Wait for the conversion (watch the dancing animation!)
4. Download your PDF when ready
5. Convert another file anytime!

## Supported File Types

- Images: JPG, PNG, GIF, BMP, WebP
- Documents: TXT, MD, CSV
- Word: DOCX
- Excel: XLSX, XLS
- HTML: HTML, HTM
- PDF: Already PDF? Just download it!

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- jsPDF
- pdf-lib
- mammoth.js (Word conversion)
- SheetJS (Excel conversion)
- html2pdf.js

## Setup

No setup required! Just open `index.html` in a web browser. All libraries are loaded from CDN.
